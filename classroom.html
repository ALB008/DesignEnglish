Aprendizajes de Clase

Etiquetas

>b: Sirve para colocar en negrita un texto

Propiedades

De la etiqueta style
>Font-family: Para definir el tipo de letra
>Font-size: Para determinar el tamaño de la letra (px,pt)
>Color: Para indicar al color de la letra
>Text -align: Para establecer la dirección del texto
>Background-color: Para definir el color del fondo

Para relizar una lista:

Lista desordenada, ordenada y descriptiva:
>ul: Define una lista desordenada
>ol: Define una lita ordenada
>li: Para definir cada elemento de una lista ordenada o desordenada
>dl: Define una lista descriptiva
>dt: Para definir el item que se desea describir
>dd: Para aregarle la descripción al item
